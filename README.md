**goodbit-mobile** - приложение для создания постов. 

## Создано на базе Expo

## Getting started
***
To get the running locally:
- Clone this repo
- `npm install` to install all required dependencies
- `npm start` to start the local server

## Available commands
***
- buid for android:
`expo build:android`
